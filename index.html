Privacy Policy

How FastTrack Jobs works

FastTrack Jobs gathers job description data from job boards like LinkedIn and uses Oauth 2.0 in order to create a google sheet and then write the job data into it.

What personal information is collected?

None, FastTrack Jobs uses Oauth 2.0 to authorize access to users’ google sheets, which means FastTrack Jobs does not collect users’ google account credentials.

How is personal information used?

According to the Oauth 2.0 Spreadsheets scope: FastTrack Jobs will be able to “see, edit create, and delete all of your google spreadsheets”. However, the sole purpose of using Oauth 2.0 to access users’ google sheets is to write job description information that was copied from job boards such as LinkedIn.

FastTrack Jobs creates a new google sheet for the user and will exclusively work in that sheet only.

FastTrack jobs WILL NOT:

  • Read information from any of the users’ personal spreadsheets
  • Edit information from any of the users’ personal spreadsheets
  • Delete any of the users’ personal spreadsheets

Does FastTrack Jobs disclose personal information to third parties?

FastTrack Jobs does not disclose any personal information to third parties.

Security of personal information

FastTrack Jobs does not collect any personal information from users, thus there are no security measures in place.

User rights to their data

FastTrack Jobs does not collect any personal data.
